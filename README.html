Projet BabyFoot

Description du projet

Mon application web permet de créer des parties de babyfoot en collaboration et en temps réel.

GitHub

Pré-requis:(Testés sous :)

  • Postgresql (11)
  • NodeJs (13.9.0)

Installation

1 Création de la BDD

  • Créez une base de donnée ‘partiesbabyfoot’.
  • Crééz une table “parties”: 
create table parties (ID SERIAL PRIMARY KEY,joueur1 varchar(250),joueur2 varchar(250),partie_termine boolean,score1 int,score2 int);

2 mise en place de l’application

Dans le repetoire du projet “Parties_de_BabyFoot", tapez:” 

$ npm install
  • Ouvrez le fichier qui est dans model/queries.js et modifiez les valeurs des
    attributs: <user>, <host>, <password> que vous avez rentrés pour la BDD ‘partiesbabyfoot’

3 Démarrage

  • A la racine du repertoire “Partie_de_BabyFoot”, tapez: 
$ npm start
